# workshop-llm

This is a Workshop repository to help you create your first LLM-powered Java application using Langchain4J!

## Introduction

Large Language Models became extremely popular in 2023 with ChatGPT, Midjourney, and many other generative applications made available for the public. It does unlock a greenfield of possibilities for new products and services that changes our daily routine!

Understanding some fundamentals of this new technology will help you keep updated and stay relevant in the AI revolution :)

## Setup your environment

For this workshop, you'll need to setup the following:

- Java 17+ (https://jdk.java.net/archive/)
- Maven (https://maven.apache.org/download.cgi)
- OpenAI Account (https://openai.com/)

## Concepts & Hands-on

### Models
Build an application that asks for GPT 3.5 model to create a Hello World example printing "Hello Langchain4J"

### Prompt Templates
Build an application that receives an input from the user and prints the response from GPT 3.5 model

### Memory
Build an application that for every user input, prints the response from GPT 3.5 model, keeping chat context aware

### Retrieval Augmented Generation
Build an applicaiton that read a doc and answer questions from the user regarding to that document

### Agents
Build an applicaiton that receives an input from the user, and for a specific command, it does print "Personal secret agent activated!"
